an option for human review; it does not conclude that a change is necessary or permitted.

## 3. Classification Vocabulary

Only the following proposal review labels are used:

- **No change currently indicated** — review evidence currently points to preserving the source;
  a later human decision could still require a separately governed action.
- **Clarification candidate** — wording or relationships could warrant future clarification.
- **Amendment candidate** — an in-place authority proposal is one option under review.
- **Successor candidate** — a separately versioned successor document is one option under review.
- **Supersession candidate** — a future explicit supersession relationship is an option under
  review; nothing is superseded by this label.
- **New ADR candidate** — an independently changeable decision may warrant a separate proposed
  ADR rather than expansion of ADR-042.
- **Freeze successor candidate** — a future versioned Freeze may be considered only after
  accepted underlying authority changes.
- **Historical-labeling candidate** — relationship metadata could be reviewed without changing
  source history.
- **Governance decision required** — available evidence is insufficient to select a change class.

Candidate actions likewise authorize nothing. Every row's current Wave 3 action is classification
review only.

## 5. Classification Interpretation

- Multiple candidate labels in one row are deliberately unselected alternatives. Only a future
  human disposition could choose among them.
- `No change currently indicated` is not a permanent no-impact ruling; it records the present
  Wave 3 evidence and preserves later review.
- `Prepare future ... proposal` is a candidate action type, not authorization to create that
  proposal in Wave 3.
- A Freeze successor candidate could only follow separately accepted underlying authority and a
  separate human decision. It does not diminish the current Freeze.
- A new ADR candidate would address a distinct decision only if a future review finds that
  ADR-042 should not absorb it. No new ADR is authorized here.
